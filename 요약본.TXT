[ 체크시트 앱 업데이트 요약 ]

1. 기본 설정 및 배포
   - 앱 이름 변경: 'CheckSheet' -> '체크시트' (한글 반영)
   - 앱 버전 업: 1.0.0+1 -> 1.0.0+2 (기존 앱 삭제 없이 업데이트 설치 가능)

2. 행 삭제 기능 고도화
   - 부분 제목 삭제 버그 수정: 이제 섹션 제목을 삭제하면 엑셀 파일에서도 완전히 제거됩니다.
   - 자동 저장 연동: 자동 저장 모드 사용 시, 행 삭제 확인 즉시 엑셀에 반영됩니다.

3. 리스트 관리 및 정렬
   - 전체 컬럼 정렬 지원: No, 품목코드, 수량, 완료 외에도 [보완, 공정, 비고] 헤더 클릭 시 정렬이 가능합니다.
   - 검색바 최적화: 검색 중 돋보기 아이콘 숨김 처리로 공간 효율 증대.

4. UI/UX 편의성 개선
   - 공정 선택창 개선: 길게 나열되던 공정 버튼을 2열 그리드(Grid)로 배치하여 한눈에 보기 편하게 수정.
   - 비밀번호 토글: PW 입력란 우측 눈 모양 아이콘으로 비밀번호 숨김/표시 제어 가능.

5. 네트워크 및 동기화 (SMB)
   - 접속 테스트 개선: 성공 시 단순 메시지가 아닌 서버 내 [공유 폴더 목록]을 함께 표시하여 연결 상태 확인 강화.
   - PDF 동기화: 원격 서버의 PDF 파일을 로컬로 안정적으로 동기화하는 로직 복구.
