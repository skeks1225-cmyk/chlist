[ 체크시트(CheckSheet) 앱 재구축 및 개발 지침서 ]

본 문서는 앱을 처음부터 다시 제작하거나 주요 기능을 복구할 때, 시행착오 없이 현재의 완성된 결과물에 도달하기 위한 핵심 기술 명세와 로직을 정리한 가이드입니다.

1. 기술 스택 및 환경 설정
   - 언어 및 프레임워크: Flutter (Dart)
   - 타겟 플랫폼: Android (권한: MANAGE_EXTERNAL_STORAGE 필수)
   - 핵심 라이브러리 (버전 준수 필수):
     * excel: ^4.0.6 (엑셀 읽기/쓰기)
     * pdfrx: 1.3.5 (PDF 렌더링 - ❗excel과의 호환성을 위해 반드시 이 버전 고정)
     * jcifs-ng: (안드로이드 네이티브 SMB 통신용 - Kotlin에서 구현)
     * shared_preferences, permission_handler, path, file_picker

2. 엑셀 데이터 로드 및 관리 로직
   - "null" 문자열 방어 (이중 보안): 엑셀 빈 셀을 읽을 때 "null" 글자가 들어오는 버그 대응을 위해 `ExcelService`와 `ItemModel` 생성자 양쪽에서 빈 값("") 치환 로직 구현 필수.
   - 가상 번호 부여 (Display Numbering): 
     * 'no'가 있는 행은 메인 번호로 인식.
     * 'no'가 비어있고 '수량'이 있는 행은 "메인번호-순번" 형태(예: 2-1, 2-2)로 가상 번호 자동 생성.
   - 부분제목(Subheading) 처리: 'no'와 '수량'이 모두 없고 '품목코드'만 있는 행을 섹션 구분선으로 인식. 해당 타이틀을 하위 품목들의 `subheadingTitle` 필드에 자동으로 전파하여 저장.

3. 리스트 화면(ChecklistScreen) 핵심 기능
   - 한글 입력 최적화: 비고(Remarks) 입력 시 한글 받침 끊김 현상을 방지하기 위해 별도의 StatefulWidget(`_RemarksCell`)로 분리 구현.
   - 포커스 해제: 입력 완료(onSubmitted) 또는 외부 터치 시 `_forgetFocus()`를 통해 키패드를 즉시 닫고 가상 포커스 노드(`_dummyFocusNode`)로 이동.
   - 정렬 로직: 모든 헤더 클릭 시 정렬 지원. 특히 가상 번호(2-10 등)는 단순 문자열 정렬이 아닌 '-' 기준 숫자 비교(`_compareDisplayNo`) 로직 적용.

4. 중앙 포커싱 및 터치 하이라이트 (외부 패키지 미사용)
   - 구현 방식: 빌드 안정성을 위해 외부 패키지 없이 순수 `ScrollController`만 사용.
   - 높이 고정 계산: 일반 항목(45px), 부분제목(40px)으로 높이를 고정하여 대상 항목의 절대 위치(offset)를 내부 로직으로 합산 계산.
   - 중앙 스크롤: 뷰어에서 돌아올 때 품목코드를 추적하여 해당 위치가 화면 정중앙(alignment 0.5)에 오도록 `animateTo` 실행.
   - 터치 기반 해제: `Listener` 위젯의 `onPointerDown`을 사용하여 자동 스크롤 중에는 테두리를 유지하고, 사용자의 실제 손가락이 화면에 닿는 순간에만 즉시 테두리(줄 전체 파란색 사각형)를 삭제.

5. PDF 뷰어 화면(PdfViewerScreen)
   - 제목 UI: AppBar를 `Column`으로 구성하여 첫 줄에 부분제목(`subheadingTitle`), 두 줄째에 품목코드를 표시하여 가독성 확보.
   - 네비게이션: 한 손 조작을 위해 [<-이전] [다음->] 버튼을 본문 좌측 하단(`bottom: 80, left: 5`)에 플로팅 형태로 배치.
   - 상태 동기화: 뷰어 내에서 변경한 완료, 보완, 공정, 비고 상태는 즉시 체크리스트 화면과 엑셀 파일(자동 저장 시)에 반영.

6. SMB 공유 폴더 연동 (Native Bridge)
   - 구조: Dart(`SmbService`)와 Kotlin(`SmbHandler`) 간의 `MethodChannel` 통신.
   - 특징: 매 호출마다 IP/ID/PW를 함께 전달하여 인증 유실 방지. 대소문자 구분 없는 파일 찾기 로직 및 타임스탬프 기반 중복 다운로드 방지 처리 적용.

7. 파일 시스템 경로
   - 기본 작업 경로: `/storage/emulated/0/Download/CheckSheet/`
   - 이 경로 내에 엑셀 파일과 PDF 도면 파일들이 위치하며, SMB 다운로드 시에도 이 폴더를 기본 저장소로 사용.
